# 🧠 Quiz App

An interactive JavaScript quiz built with Vanilla JavaScript and ES6 Modules.

## ✨ Features

- 10 JavaScript questions
- Instant feedback (correct/wrong highlighting)
- Progress bar and question counter
- Score and percentage at the end
- High score saved locally
- Shuffled questions every game

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- ES6 Modules (import/export)
- Destructuring & Spread operator
- localStorage
